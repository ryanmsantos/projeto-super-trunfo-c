# 🎮 Projeto Super Trunfo em C

Este projeto foi desenvolvido como parte da minha evolução na linguagem C durante o curso de Análise e Desenvolvimento de Sistemas.

A proposta foi construir um sistema inspirado no jogo Super Trunfo, evoluindo o nível de complexidade em três etapas: Novato, Aventureiro e Mestre.

---

## 📌 Estrutura do Projeto

O projeto está dividido em três níveis progressivos:

### 🔹 01 - Nível Novato
- Cadastro de duas cartas
- Uso de variáveis `char` e `int`
- Entrada e saída de dados com `scanf` e `printf`
- Organização básica de código

### 🔹 02 - Nível Aventureiro
- Introdução ao tipo `float`
- Cálculo de:
  - Densidade populacional
  - PIB per capita
- Formatação de casas decimais
- Evolução na lógica matemática

### 🔹 03 - Nível Mestre
- Uso de `unsigned long int`
- Cálculo de Super Poder
- Comparação atributo por atributo
- Uso de operadores relacionais
- Estrutura mais robusta de lógica
